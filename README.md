**How to run this thing?**<br />

  Using docker-compose:<br />
  ```
  git clone https://github.com/Allexandere/QuotesApp.git
  cd QuotesApp/
  mvn clean package -DskipTests
  docker-compose up
  ```
  **Endpoints:**<br />
 
  `http://localhost:8080/`
